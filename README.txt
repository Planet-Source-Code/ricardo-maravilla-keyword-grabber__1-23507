Program Name: Keyword Grabber
Author: Misael Pateyro
Email: papitas66@hotmail.com
Date Uploaded to PSC: 5/26/01

Description: This program is designed to help people out there who have a website and would like to rank better in the search engines in order to get more traffic. After you type into the search box what someone looking for your web page might type, type in how many responses you'd like to get. Click on the search button. The program should then return the top urls  for the words you typed in. To get the keywords for a site you can now just click on the url you want and then click on the button that says "Get Keywords" or you can just drag that URL onto the listbox on the bottom left of the form and the process of getting the keywords will begin automatically. Keywords will appear in that listbox. You can then just drag those keywords that you want to save for future use onto the grid that says "My Keywords" on the bottom right. The keywords in "My Keywords" will be saved in the file MYKWs.txt and this is also checked by the program when it starts so that you can view your saved words. When you want to remove words from "My Keywords" you can drag them off and drop them somewhere else on the form of just click on them and press the remove button. To generate the html code for a web page just click on the button "Generate HTML Code" and paste it into your HTML file.
